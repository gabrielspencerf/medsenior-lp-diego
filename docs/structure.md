# Estrutura do Projeto - BrasCare MedSênior

Este projeto foi desenvolvido como uma landing page de alta performance e alto padrão visual para a BrasCare, focada nos planos MedSênior em Curitiba.

## Stack Tecnológica
- **React 18**: Framework para construção da interface.
- **Vite**: Ferramenta de build e servidor de desenvolvimento ultra-rápido.
- **Tailwind CSS**: Framework de estilização utilitária para design consistente e responsivo.
- **Framer Motion**: Biblioteca para animações avançadas e interações fluidas (como o carrossel de planos).
- **Lucide React**: Biblioteca de ícones vetoriais modernos.

## Arquitetura de Pastas
- `/src/components`: Componentes reutilizáveis (UI).
- `/src/App.tsx`: Ponto de entrada principal contendo a lógica de seções e estados.
- `/src/index.css`: Arquivo global de estilos e definições do Tailwind.
- `/docs`: Documentação técnica.
- `/docs/design`: Regras do Design System.
- `/docs/copy.md`: Estratégia de copywriting.
- `/docs/maintenance.md`: Boas práticas e regras de atualização das docs.
- `/docs/logs`: Registro de problemas identificados e soluções aplicadas.

## Funcionalidades Principais
1. **Hero Section Dinâmica**: Entrada com animações de opacidade e escala.
2. **Carrossel Interativo**: Sistema de planos com suporte a `drag` e navegação via botões.
3. **Escaneabilidade Refinada**: Hierarquia visual baseada em pesos de fonte e espaçamentos generosos (80px - 128px entre seções).
4. **Responsividade Total**: Adaptado para dispositivos móveis, tablets e desktops ultra-wide.
