# Claude — regras e contexto

## Resumo

Projeto **medsenior-lp-enanda**: página de captação BrasCare / MedSênior Curitiba (**React 19** + Vite + Tailwind v4 + Motion). Código principal da LP: `src/pages/LandingPage.tsx`; rotas: `src/App.tsx`; estilos: `src/index.css`; entrada: `src/main.tsx`.

## Instruções de trabalho

1. Trate `docs/README.md` como índice; confirme estrutura real em `docs/structure.md` (a árvore de pastas pode mudar).
2. Alterações de copy devem refletir também em `docs/copy.md` na mesma tarefa, quando forem estratégicas (não só typos).
3. Alterações visuais recorrentes (novos tokens, novo padrão de card): atualizar `docs/design/design-system.md`.
4. Use o formato de log completo em `docs/logs/fixes.md` após corrigir defeitos ou regressões.

## Estilo de resposta (sugerido)
