# Assistentes de IA neste repositório

Documentação para quem edita o projeto via **Cursor**, **Claude** ou **GPT** (ChatGPT, Copilot, etc.), sem depender de memória de conversa.

## Checklist comum (todas as ferramentas)

1. Ler `docs/README.md` (mapa), depois `docs/structure.md`, `docs/design/design-system.md` e `docs/copy.md` antes de mudanças visuais ou de texto.
2. Seguir `docs/maintenance.md` para saber o que atualizar na mesma tarefa.
3. Registrar em `docs/logs/fixes.md` qualquer correção relevante com **Problema**, **Causa**, **Solução**, **Prevenção**.
4. Preferir mudanças pequenas e focadas; não refatorar arquivos inteiros sem pedido explícito.
5. Manter coerência com a LP: tom premium, foco Curitiba / 44+, CTAs para WhatsApp.

## Arquivos por ferramenta

- [cursor.md](./cursor.md) — uso no Cursor IDE / Agent.  
- [claude.md](./claude.md) — uso no Claude (web ou Code).  
- [gpt.md](./gpt.md) — uso no ChatGPT / ferramentas similares.

O resumo técnico do produto continua centralizado em `docs/README.md` (mapa da pasta `docs`).
